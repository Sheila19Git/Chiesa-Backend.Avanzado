 # Entrega N°1 – Backend con Node.js y Express

## Descripción
Servidor backend desarrollado con Node.js y Express para la gestión de productos y carritos de compra.  
La información se persiste utilizando archivos JSON.

---

## Tecnologías utilizadas
- Node.js
- Express
- Thunder Client / Postman (para pruebas)
